# ERC1155 Boilerplate
erc1155 sample/example implementation

## Installation

```shell
yarn install # or npm install
```

## Deploy and verify

MyERC1155 example
```shell
# modify package.json
yarn run deploy:mumbai
yarn run vefiry:mumbai
```

Metadata example
```shell
brew install ipfs # if not installed
yarn run ipfs:add
yarn run ipfs:cat
yarn run ipfs:open
```
